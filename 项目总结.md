## 已实现功能总览

### 登录页

- 邮箱 + 密码登录表单
- 微信/QQ/GitHub 第三方登录图标（点击提示"暂未开通"）
- 非预设账号登录提示"该账号未注册"
- 密码错误提示"密码错误"
- 登录成功根据角色自动跳转

### 学生端 (`/student`)

- 浏览全部 24 门课程
- 搜索框：按课程名称或编号关键词搜索
- 学期下拉筛选
- 学院下拉筛选
- 四种标签切换：全部课程 / 已选修 / 已评价 / 待评价
- 已评价课程：绿色"已评价（查看）"按钮 + "详情"按钮弹出弹窗（五星展示 + 留言原文）
- 已选修未评价课程：蓝色"去评价"按钮
- 未选修课程：灰色"未选修"锁定态
- 顶部"我的评价历史"按钮：查看自己所有评价记录
- 评价表单：五维度星级评分（教学内容/教学态度/教学方法/考核方式/综合满意度）+ 500字留言 + 防重复提交

### 教师端 (`/teacher`)

- 数据看板：
  - 评分分布柱状图（1-5 分频次）
  - 五维雷达图
  - 课程对比趋势折线图
  - 评价词云
  - 顶部指标卡片（教授课程数/评价总数/综合平均分/留言总数）
  - 各维度详细统计表格（平均分/中位数/标准差/最高/最低/评价数）
- 按课程切换标签，图表联动更新
- 留言明细页面：按课程筛选，逐条查看学生姓名/学号/日期/五维分数/留言原文

### 管理员端 (`/admin`)

- 数据概览：课程评价排名条形图 + 各课程评分卡片
- 课程管理：24 门课程完整表格
- 评价记录：所有评价列表 + 留言原文

## API 接口

| 方法 | 路径 | 功能 |
|------|------|------|
| POST | /api/auth/login | 用户登录 |
| POST | /api/auth/logout | 用户登出 |
| GET | /api/auth/me | 获取当前用户信息 |
| GET | /api/courses | 获取课程列表（含选课/评价状态） |
| POST | /api/evaluations | 提交评价（防重复+权限校验） |
| GET | /api/stats/dashboard | 获取统计数据（按角色返回不同数据） |

```
Supabase PostgreSQL（数据库）
```

## 本地开发

```bash
# 进入项目目录
cd course-eval

# 安装依赖
npm install

# 生成 Prisma 客户端（需要在 prisma/schema.prisma 中把 provider 改为 sqlite）
npx prisma generate

# 初始化本地 SQLite 数据库并填充种子数据
npx tsx prisma/seed.ts

# 启动开发服务器
npm run dev

# 浏览器打开 http://localhost:3000
```

## 修改代码后的部署流程

1. 本地修改代码
2. `git add -A && git commit -m "描述" && git push`
3. Netlify 自动检测到 GitHub 推送，自动重新构建部署
4. 等待约 2-3 分钟，网站自动更新
